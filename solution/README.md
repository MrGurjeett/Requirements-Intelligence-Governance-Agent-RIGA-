# Solution Package

## Contents
Place your exported Copilot Studio solution here:
- EnterpriseRequirementsAgent.zip

## How to Export
1. Go to make.powerapps.com
2. Click Solutions
3. Create new solution and add your agent
4. Click Export → Managed
5. Place the downloaded .zip file in this folder

## How to Import
1. Go to make.powerapps.com
2. Select target environment
3. Click Solutions → Import Solution
4. Upload the .zip file from this folder
